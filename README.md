## UserBlog

基于 Express + mongoDB 的个人微博发布平台

## 开发环境
  - node.js : v6.10.1
  - mongoDB : v2.6.7
  - express : v3.0.0
  - linux   : v3.10.0-123.el7.x86_64 (builder@kbuilder.dev.centos.org) (gcc version 4.8.2 20140120 (Red Hat 4.8.2-16) (GCC) )
